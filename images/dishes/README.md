# Фотографии блюд

Эта папка содержит изображения блюд для меню ресторана "Жамэль".

## Требования к фотографиям:
- Формат: JPG
- Рекомендуемый размер: 800x600px или больше
- Соотношение сторон: 4:3 или 16:9
- Качество: высокое (не менее 80%)

## Названия файлов:
Названия файлов должны соответствовать названиям в `menu-data.js`.
Используйте транслитерацию: "Овощной букет" → `ovoshnoy-buket.jpg`

## Пример:
- ovoshnoy-buket.jpg
- zelenaya-gryadka.jpg
- assorti-syrov-armeniya.jpg
- teplyy-salat-zhamel.jpg
- spas.jpg
- hashlama.jpg
- dolma.jpg
- hanuma.jpg

## Placeholder:
Файл `placeholder.svg` используется автоматически для блюд, у которых ещё нет фотографии.
