# 🇦🇲 Веб-меню армянского ресторана

Красивое, адаптивное веб-меню с QR-кодом для доступа гостей.

---

## 📸 Как загрузить фотографии блюд

> Фотографии хранятся в папке **`images/dishes/`** внутри этого репозитория.

### Шаг 1 — Найдите папку `images/dishes/` на GitHub

1. Откройте ваш репозиторий: **https://github.com/9810artur/menu**
2. На главной странице репозитория кликните на папку **`images`**
3. Затем откройте папку **`dishes`**

Вы попадёте сюда:
```
https://github.com/9810artur/menu/tree/main/images/dishes
```

### Шаг 2 — Загрузите фотографии

1. Находясь в папке `images/dishes`, нажмите кнопку **"Add file"** → **"Upload files"**
2. Перетащите ваши JPG-файлы или нажмите **"choose your files"**
3. Внизу страницы нажмите **"Commit changes"**

### Шаг 3 — Правильные названия файлов

Каждое фото должно называться точно так, как указано в файле `menu-data.js`.  
Используется транслитерация русских названий блюд:

| Блюдо | Имя файла |
|-------|-----------|
| Овощной букет | `ovoshchnoy-buket.jpg` |
| Зелёная грядка | `zelyonaya-gryadka.jpg` |
| Спас | `spas.jpg` |
| Хашлама | `khashlama.jpg` |
| Долма | `dolma.jpg` |
| Тёплый салат Жамэль | `tyoplyy-salat-zhamel.jpg` |

> **Совет:** Если не знаете точное имя файла для блюда — откройте `menu-data.js`  
> и найдите поле `image:` рядом с нужным блюдом.

### Требования к фотографиям

- Формат: **JPG**
- Размер: **800×600 px** или больше
- Качество: **не менее 80%**

## 🚀 Быстрый старт

### Шаг 1: Активируйте GitHub Pages

1. Зайдите в **Settings** репозитория
2. Выберите **Pages** в левом меню
3. В разделе **Source** выберите ветку **main**
4. Нажмите **Save**
5. Через 1-2 минуты ваше меню будет доступно по адресу:
   ```
   https://9810artur.github.io/menu/
   ```

### Шаг 2: Создайте QR-код

1. Откройте страницу генератора: https://9810artur.github.io/menu/qr-code.html
2. Введите URL вашего меню: https://9810artur.github.io/menu/
3. Нажмите **Generate QR Code**
4. Скачайте изображение
5. Распечатайте (минимум 5x5 см)
6. Разместите на столах ресторана

### Шаг 3: Редактируйте меню

1. Откройте файл `menu-data.js` в репозитории
2. Измените названия блюд, описания и цены
3. Сохраните файл — меню обновится автоматически!

```

│       ├── README.md         (инструкция по именованию)
│       └── ovoshchnoy-buket.jpg  (и т.д.)
└── README.md           — Эта инструкция
```

## ✏️ Как редактировать меню

Откройте файл `menu-data.js` и измените данные. Каждое блюдо выглядит так:

```javascript
{
  name: 'Овощной букет',
  weight: '450 г',
  description: 'Помидоры, огурцы, острый перец, зелень.',
  price: 870,
  emoji: '🥗',
  image: 'images/dishes/ovoshchnoy-buket.jpg'   // ← имя вашего JPG-файла
}
```

## 📱 Как это работает?

1. Гость сканирует QR-код на столе
2. Открывается красивое меню на его телефоне
3. Он видит фотографии блюд, описания и цены
4. Может искать блюда — готово!
